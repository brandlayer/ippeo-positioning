01 / Problem

일본 환자의 핵심 문제 3가지

일본 환자는 한국 병원 정보가 부족해서가 아니라, 결정 전 불안이 해소되지 않기 때문에 망설입니다.

문제 ①

어떤 병원이 맞는지 모르겠다

후기와 가격은 많지만, 내 상태에 맞는 병원인지 판단하기 어렵습니다.

문제 ②

언어가 안 통한다

내 고민이 정확히 전달되는지, 의사의 설명을 제대로 이해하는지 불안합니다.

문제 ③

한국까지 가서 실패하면 어쩌나

시술 후 귀국하면 문제가 생겨도 연결이 끊길 수 있다는 불안이 큽니다.

02 / Solution

이뻐가 해결하는 것

이뻐는 병원 리스트를 보여주는 데서 끝나지 않고, 환자가 실제로 결정을 내리는 순간까지 연결합니다.

일본 환자의 불안이뻐의 해결고객이 느끼는 가치
병원 선택 불안검증된 병원의 의사가 직접 상담하여 시술 적합 여부를 판단“내 상태를 보고 말해주는구나”
언어 장벽실시간 통역 MediLinQ로 의사와 환자 간 직접 소통 지원“내 말이 제대로 전달되는구나”
귀국 후 단절비대면 사후상담과 병원 재연결 구조 제공“시술 후에도 혼자가 아니구나”

이뻐의 본질은 병원 검색이 아니라
한국 미용의료 결정을 안심하게 만드는 것입니다.

Positioning Core
03 / Journey

이뻐가 책임지는 5단계 여정

고객의 불안은 한 번에 사라지지 않습니다. 결정 전, 상담, 예약, 방한, 귀국 후 각 단계마다 다른 불안을 해소해야 합니다.

결정 전

검증된 병원의 의사가 직접 상담하여 환자 상태와 시술 적합 여부를 판단합니다.

비교가 아니라 확정에 가까워집니다.

상담

실시간 통역으로 의사와 환자가 직접 연결됩니다. 에이전트 필터링을 줄입니다.

의사의 말이 왜곡 없이 전달됩니다.

예약·결제

예약금, 환불 기준, 일정 안내를 투명하게 정리합니다.

결제 불안을 줄입니다.

방한·시술

현지 코디네이터와 병원 연결을 통해 환자가 혼자 대응하지 않도록 지원합니다.

현장 불안을 줄입니다.

귀국 후

귀국 후 비대면 사후상담과 담당 병원 재연결을 지원합니다.

단절되지 않는 여정입니다.

04 / Differentiation

경쟁사 대비 차별화 3가지

이뻐의 차별성은 기능 하나가 아니라, 의사 직접성·언어 해결·귀국 후 연결이 결합된 신뢰 구조입니다.

차별화 ①

의사 직접 연결

기존 에이전트는 환자와 의사 사이에 전달자가 있습니다. 이뻐는 환자가 의사의 판단을 직접 듣는 구조를 지향합니다.

차별화 ②

실시간 통역 내장

병원 일본어 직원 역량에 의존하지 않고, 의료 특화 통역 품질을 플랫폼 안에 내장합니다.

차별화 ③

귀국 후 사후관리

경쟁사가 끊기는 시점 이후에도 담당 병원과 재연결되는 구조를 만듭니다.

구분기존 에이전트병원 SNS/플랫폼이뻐
상담 구조에이전트 중간 전달병원별 직접 문의의사 직접 상담 + 통역
언어 해결상담자 역량 의존병원 직원 역량 의존MediLinQ 기반 표준화
예약 이후귀국 후 단절 가능예약/시술 후 단절귀국 후 연결 지향
고객 가치편의성정보 탐색결정 안심
05 / Execution

현재 실행 가능 여부와 우선순위
